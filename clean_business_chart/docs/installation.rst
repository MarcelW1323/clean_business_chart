============
Installation
============


Stable release
--------------

To install Clean Business Chart, run this command in your terminal:

.. code-block:: console

    $ pip install clean_business_chart

This is the preferred method to install Clean Business Chart, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Clean Business Chart can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/marcelw1323/clean_business_chart

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/marcelw1323/clean_business_chart/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/marcelw1323/clean_business_chart
.. _tarball: https://github.com/marcelw1323/clean_business_chart/tarball/master
